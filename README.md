Submission Belajar Fundamental Front-End Web Development (Dicoding) <br>
CARA INSTALASI

1. Jalankan perintah<br>
   `npm install`
2. Menjalankan aplikasi menggunakan webpack production<br>
   `npm run build`
3. Atau menggunakan webpack development<br>
   `npm run start-dev`
